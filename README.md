# Global Layoffs Analysis Dashboard

## Overview
This project presents an interactive Tableau dashboard analyzing global layoffs across companies, industries, countries, and time.

## Tools Used
- Tableau
- CSV Dataset
- GitHub

## Key Insights
- Layoff trends over time
- Most affected industries
- Top companies by layoffs
- Country-wise layoff distribution

## Files
- `data/layoffs.csv` – Dataset used
- `tableau/Global Layoffs Analysis Dashboard.twbx` – Tableau packaged workbook
- `images/dashboard_preview.png` – Dashboard snapshot
